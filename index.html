系统说明

SQLite + Static Web

为满足“提供一个可以直接访问的网址、无需后端运行环境”的要求，本项目采用静态网页 + 浏览器内 SQLite 数据库的实现方式，部署到任意静态托管平台即可在线访问。

数据库包含 user、item、orders 三张表，并在页面中完成插入、修改、删除、连接查询、聚合统计、视图查询和事务购买。
